# Vincent Gbafah Portfolio

A personal portfolio website showcasing skills and projects.

## 📋 About

This is a clean, responsive portfolio website built with HTML and CSS. It serves as a digital resume and project showcase for Vincent Gbafah.

## 🛠️ Technologies

- **HTML** (46.9%) - Semantic markup and page structure
- **CSS** (53.1%) - Styling and responsive design

## 📁 Project Structure

```
vincent-gbafah-portfolio/
├── index.html          # Main portfolio page
└── styles.css          # Styling and layout
```

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/vincentgbafah/vincent-gbafah-portfolio.git
cd vincent-gbafah-portfolio
```

2. Open `index.html` in your web browser to view the portfolio

## 📦 Deployment

This repository is configured with GitHub Pages. The portfolio is automatically deployed and accessible online.
